# AquaSave
A website that keeps on in track with water flow sensor in the main pipe of the house,alerts the conserned whenever  water leak is detected.The website has various features like live tracking ,Gaming and dashboard analysis for easy use.
The webiste includes:
Smart Dashboard - Real-time water usage with animated stats, weekly bar charts, daily flow patterns, and comparison metrics
Live Monitoring - Animated water drop with pulsing rings when flow is detected, real-time flow rate updates, and activity logs
Leak Detection Scenario - Interactive 3-stage demonstration (Normal → Increasing Flow → Alert) with detailed anomaly visualization and actionable recommendations
AI Insights - Smart suggestions with priority levels, predictive analytics, weekly pattern analysis, and personalized tips
Impact & Gamification - Points/levels system, achievements, environmental impact tracking, cost savings, and community leaderboard
System Architecture - Complete technical flow diagram, component details, data flow process, and technology stack overview
